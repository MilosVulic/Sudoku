# Sudoku
### Sudoku 2D game in java with javaFX GUI components


List of functionalities:

1. **Animated User manuals**
2. **Game difficulty**
3. **Accuracy check**
4. **Solve all**
5. **Solve one field**
6. **Clearing all fields**
7. **Clear one**
8. **Create your own sudoku depending on your inputs**
